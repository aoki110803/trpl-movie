# TRPL Movie

Aplikasi pencarian film sederhana menggunakan HTML, CSS, dan JavaScript (jQuery) dengan data dari OMDb Public API.

Dibuat untuk memenuhi tugas praktikum Sistem Terdistribusi Part 2 - Public API / Open API.

## Struktur File

```
trpl-movie/
├── index.html
└── js/
    └── script.js
```

## Cara Menjalankan

Buka file index.html langsung di browser, atau letakkan folder di dalam direktori web server (contoh: laragon/www/) lalu akses melalui localhost.

## API yang Digunakan

Sumber data: https://www.omdbapi.com

- Pencarian film: parameter `s` berdasarkan judul
- Detail film: parameter `i` berdasarkan imdbID
